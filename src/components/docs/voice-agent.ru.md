# VOICE_AGENT — обзор

*EN version: [VoiceAgentDocs.astro](https://github.com/nikitak-dev/nikitak-dev.github.io/blob/main/src/components/docs/VoiceAgentDocs.astro)*

---

### // WHAT IT DOES

Голосовой AI-ресепшнист отвечает на каждый входящий звонок в естественном разговоре, превращает его в забронированную, перенесённую или отменённую встречу в реальном календаре и фиксирует итог — без участия человека.

### // HOW IT WORKS

Узнаёт, кто звонит, отвечает на вопросы об услугах, ценах и часах работы из фиксированной базы знаний и проговаривает каждое бронирование для подтверждения, прежде чем его зафиксировать. По окончании звонка расшифровка, итог и аудиозапись сохраняются автоматически в карточке звонящего.

### // GOOD TO KNOW

Проект в разработке, показан на вымышленном бизнесе (GreenScape Landscaping). Агент не на публичной линии — живые звонки дороги в эксплуатации, поэтому проект представлен здесь, а не доступен для дозвона.
