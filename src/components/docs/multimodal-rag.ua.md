# MULTIMODAL_RAG — огляд

### // WHAT IT DOES

Запитуйте природною мовою й отримуйте пряму відповідь із бібліотеки різноформатних файлів — текст, зображення, PDF, аудіо, відео. До кожної відповіді додаються клікабельні джерела, щоб відкрити саме той файл, звідки взято відповідь — аж до конкретного кадру відео чи сторінки PDF.

### // TRY IT

Запустіть живу демонстрацію на цій сторінці: оберіть підказаний запит або введіть свій, натисніть EXEC і побачите відповідь із джерелами.

### // GOOD TO KNOW

Бібліотека — фіксований демонабір, який веде розробник. По ньому можна шукати, але додати власні файли не можна — завантаження нових документів у пошукову (векторну) базу виконує лише розробник, відвідувачам воно недоступне.
