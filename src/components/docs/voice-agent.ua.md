### // MEET SOPHIE

Sophie — голосова AI-рецепціоністка цього проєкту, той самий голос, який ви почуєте під час дзвінка.

### // WHAT IT DOES

Sophie приймає вхідні дзвінки й веде їх у природній розмові: упізнає того, хто телефонує, відповідає на запитання про послуги, ціни та години роботи з фіксованої бази знань, бронює, переносить або скасовує зустрічі в реальному календарі — проговорюючи деталі для підтвердження. Кожен дзвінок зберігається автоматично (розшифровка, підсумок, аудіозапис).

### // HEAR IT

Увімкніть запис демонстраційного дзвінка на цій сторінці й простежте повне бронювання від привітання до підтвердження.

### // GOOD TO KNOW

Демонстрація на вигаданому бізнесі (GreenScape Landscaping). Агент не на публічній лінії — те, що ви чуєте, це запис справжнього дзвінка.
