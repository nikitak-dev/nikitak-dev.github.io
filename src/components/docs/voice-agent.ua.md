# VOICE_AGENT — огляд

### // WHAT IT DOES

Голосовий AI-рецепціоніст відповідає на кожен вхідний дзвінок у природній розмові, перетворює його на заброньовану, перенесену або скасовану зустріч у реальному календарі й фіксує підсумок — без участі людини.

### // HOW IT WORKS

Упізнає, хто телефонує, відповідає на запитання про послуги, ціни та години роботи з фіксованої бази знань і проговорює кожне бронювання для підтвердження, перш ніж його зафіксувати. Після завершення дзвінка розшифровка, підсумок та аудіозапис зберігаються автоматично в картці того, хто телефонує.

### // GOOD TO KNOW

Проєкт у розробці, показаний на вигаданому бізнесі (GreenScape Landscaping). Агент не на публічній лінії — живі дзвінки дорогі в експлуатації, тому проєкт представлено тут, а не доступний для дзвінка.
