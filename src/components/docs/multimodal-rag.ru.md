# MULTIMODAL_RAG — обзор

*EN version: [MultimodalRagDocs.astro](https://github.com/nikitak-dev/nikitak-dev.github.io/blob/main/src/components/docs/MultimodalRagDocs.astro)*

---

### // WHAT IT DOES

Спрашивайте на естественном языке и получайте прямой ответ из библиотеки разноформатных файлов — текст, изображения, PDF, аудио, видео. К каждому ответу прилагаются кликабельные источники, чтобы открыть тот самый файл, откуда взят ответ — вплоть до конкретного кадра видео или страницы PDF.

### // TRY IT

Запустите живую демонстрацию на этой странице: выберите подсказанный запрос или введите свой, нажмите EXEC и прочитайте ответ с источниками.

### // GOOD TO KNOW

Библиотека — фиксированный демо-набор, который ведёт разработчик. По нему можно искать, но добавить свои файлы нельзя — загрузку новых документов в поисковую (векторную) базу выполняет только разработчик, посетителям она недоступна.
